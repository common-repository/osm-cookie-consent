== Description ==

This plugin allows you to inform your users about the UE Cookies Law.

It is a lightweight JavaScript plugin (silktide based) for alerting users about the use of cookies on your website.

Fully responsive and 100% customizable. You can writte you own info text, more info link, colors, bacground colors, font-size, etc...

WPML compatible. Polylang compatible.


== Installation ==

Extract the zip file and just drop the contents in the <code>wp-content/plugins/</code> directory of your WordPress installation (or install it directly from your dashboard) and then activate it from Plugins page.
